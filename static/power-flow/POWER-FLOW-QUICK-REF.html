⚡ Power Flow Component - Quick Reference

Complete API reference and code examples

🚀 Quick Start

// 1. Add script tag to HTML <script src="power-flow-component.js"></script> // 2. Create container <div id="svg-container"></div> // 3. Initialize component const component = new PowerFlowComponent('svg-container', { isPluggedIn: false, batteryLevel: 75, amperageMa: -1200 });

Constructor

new PowerFlowComponent(containerId, initialState)

Full State Object Properties:

Property Type Description
isPluggedInbooleanIs wall power connected?
batteryLevelnumberCharge level (0-100)
isChargingbooleanIs battery actively charging?
powerFlowIntensitynumberAnimation speed (0.2-2.0)
voltageMvnumberBattery voltage in millivolts
amperageManumberCurrent flow (+ in, - out) in mA
powerWnumberBattery power in Watts
systemPowerWnumberTotal system usage in Watts
adapterWattsnumberCharger capacity in Watts
temperatureCnumberBattery temperature in °C
cycleCountnumberTotal battery discharge cycles
timeRemainingMinnumberEst. minutes remaining
sampledAtstring/DateISO timestamp of last update

Methods

METHODsetState(newState)

component.setState({ batteryLevel: 85, isCharging: true, amperageMa: 3200 });

Updates component state and refreshes visuals. Merge with existing state.

METHODgetState()

const current = component.getState();

Returns current state object.

METHODdestroy()

component.destroy();

Removes component and cleans up DOM.

Visual Behavior Guide

Condition Visual Outcome
isPluggedIn: false Yellow flow: Battery → Laptop
isPluggedIn: true Blue flow: Adapter → Laptop
isCharging: true Green flow: Laptop → Battery
batteryLevel <= 20 RED ring (Critical)
batteryLevel <= 55 YELLOW ring (Warning/Holding)
batteryLevel > 55 GREEN ring (Healthy)
